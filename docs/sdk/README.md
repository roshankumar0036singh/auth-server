## Documentation Principles

Every SDK guide in this section is organized around the same questions:

1. What must be installed and configured?
2. What is the smallest working integration?
3. How are sessions, errors, and runtime differences handled?
4. What is the complete public API?
5. Which behavior is implemented today, and which behavior requires additional application code?

> [!NOTE]
> SDK guides describe the source code in this repository. Published package versions can differ from the current checkout. Check the package version and release notes before upgrading a production application.
